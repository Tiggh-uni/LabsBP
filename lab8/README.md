# News Scraper — Веб-скрейпинг новостных заголовков

## Описание

**News Scraper** — это десктопное приложение с графическим интерфейсом (GUI) для автоматического сбора заголовков новостей с веб-сайтов. Программа загружает HTML-страницу по указанному URL, парсит её с помощью библиотек `requests` и `BeautifulSoup`, извлекает заголовки и ссылки на полные новости, а затем отображает результат в удобной таблице. Пользователь может сохранить полученные данные в текстовый файл.

Приложение реализовано с использованием **объектно-ориентированного подхода** и содержит **собственные классы исключений** для обработки ошибок сети, некорректного URL и проблем с парсингом. Длительные операции выполняются в отдельном потоке, благодаря чему интерфейс остаётся отзывчивым.

## Инструкции по запуску

### 1. Установка зависимостей

Убедитесь, что у вас установлен Python версии 3.8 или выше. Затем установите необходимые библиотеки:

pip install requests beautifulsoup4
Примечание: Tkinter входит в стандартную поставку Python, дополнительно устанавливать его не нужно.

2. Подготовка файлов
Скопируйте код приложения в два файла:

scraper.py — модуль с логикой парсинга и исключениями.

main.py — GUI на Tkinter.

3. Запуск
Откройте терминал (командную строку) в папке с проектом и выполните:

bash
python main.py
После запуска откроется главное окно приложения.

Краткая справка
Основные элементы интерфейса
Поле ввода URL – укажите адрес новостного сайта (по умолчанию подставлен https://www.bbc.com/news).

Кнопка «Получить заголовки» – запускает процесс скрейпинга. Во время загрузки кнопка блокируется, в статусной строке отображается прогресс.

Таблица результатов – содержит три колонки: номер, заголовок новости, ссылка. Длинные ссылки автоматически обрезаются для удобства просмотра.

Кнопка «Сохранить в файл» – открывает диалог сохранения, позволяя записать все заголовки и ссылки в текстовый файл (кодировка UTF-8). Становится активной только после успешного получения данных.

Кнопка «Очистить» – удаляет все строки из таблицы и сбрасывает сохранённые данные.

Статусная строка – отображает текущее состояние: «Готов», «Загрузка...», количество полученных заголовков или сообщение об ошибке.
